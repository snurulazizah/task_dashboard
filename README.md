# 📊 Task Management Dashboard System

## Overview
Interactive dashboard to manage and visualize tasks, integrated with Qontak CRM API for real-time workflow monitoring.

## Features
- Task tracking & management  
- Interactive dashboard  
- Qontak CRM API integration  
- Status categorization (To Do, In Progress, Done)  

## Tech Stack
Python, Streamlit, Plotly/Matplotlib, Qontak API  

## Setup
```bash
git clone https://github.com/sitinurulaz/task_dashboard.git
cd task_dashboard
pip install -r requirements.txt
streamlit run app.py

API Setup (Streamlit Cloud)

Go to Settings → Secrets and add:

QONTAK_API_KEY = "your_api_key_here"
