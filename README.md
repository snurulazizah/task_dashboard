# 📊 Task Management Dashboard System

## 📌 Overview
Task Management Dashboard System is an interactive application designed to manage, track, and visualize tasks efficiently through a user-friendly dashboard. The system integrates with Qontak CRM API to retrieve and process real-time CRM data for monitoring business workflows.

## 🚀 Features
- 📋 Task creation and management  
- 📊 Interactive dashboard visualization  
- 🔗 Integration with Qontak CRM API  
- ✅ Task status categorization (To Do, In Progress, Done)  
- 🔍 Real-time monitoring of CRM activities  
- 🎯 Clean and intuitive user interface  

## 🛠️ Tech Stack
- Programming Language: Python  
- Framework: Streamlit  
- Data Visualization: Plotly / Matplotlib  
- API Integration: Qontak CRM API

## ⚙️ Installation & Deployment

### 1. Clone Repository
```bash
git clone https://github.com/sitinurulaz/task_dashboard.git
cd task_dashboard
2. Install Dependencies
pip install -r requirements.txt
3. Configure API Credentials (Streamlit Cloud)

This project uses Streamlit Secrets Management instead of .env:

Deploy via https://share.streamlit.io/
Open your project
Go to Settings → Secrets
Add your API key:
QONTAK_API_KEY = "your_api_key_here"
4. Run the App Locally
streamlit run app.py
🔗 API Documentation
